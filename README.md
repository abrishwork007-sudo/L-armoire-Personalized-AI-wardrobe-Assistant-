# L-armoire-Personalized-AI-wardrobe-Assistant-
L'Armoire is an AI-powered wardrobe assistant built as a semester project . It helps users get outfit recommendations by analyzing their body silhouette and wardrobe items, with three different implementations: a web app, a terminal-based program, and a hardware prototype.

# Features
- AI-driven outfit suggestions powered by the Claude API
- Body silhouette extraction using Canvas-based image processing
- Color detection for wardrobe items via a hardware sensor
- Multiple interfaces: web, command-line, and physical device

# Project Versions

# 1. Web App (React + Vite)
The main version of L'Armoire. Uses Canvas to extract the user's body silhouette from an uploaded image, then sends data to the Claude API to generate outfit recommendations.

 Tech: React, Vite, Canvas API, Claude API

 # 2. Terminal Version (Python)
A command-line implementation of the same core logic, built entirely with object-oriented principles (classes, inheritance, encapsulation) to demonstrate OOP concepts taught in the course.

Tech: Python (OOP)

# 3. Hardware Prototype (Arduino)
A physical prototype that uses a TCS3200 color sensor to detect the color of a clothing item, displaying results on an LCD screen.
